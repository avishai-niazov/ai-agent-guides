# Agent Architecture Guide

A practical 4-page guide for building, reviewing, and fixing AI agents.

This guide explains:

- What a proper AI agent must include
- Which architecture components are required
- Which components are optional but useful as agents grow
- How to review an existing agent
- How to generate missing governance files around an agent

## Suggested use

1. Read the PDF guide first.
2. Run the review prompt against your existing agent files.
3. Read the review report.
4. Run the fix prompt only after the review is complete.
5. Test the generated files before sharing or deploying the agent.
