# AI Agent Guides

Copyable prompts, visual guides, and practical resources for building, reviewing, and improving AI agents.

This repository supports public posts, LinkedIn carousels, QR-code links, and reusable AI-agent resources.

## Guides

| Guide | Purpose | Status |
|---|---|---|
| [Agent Architecture Guide](./agent-architecture-guide/) | Defines the minimum structure of a proper AI agent and provides copyable review/fix prompts | Published |

## How to use this repository

Use the visual guide or carousel for quick reading, then open the related prompt files to copy the text directly.

Each guide folder may include:

- `README.md` — guide overview and usage notes
- `prompts/` — copyable prompt files
- `pdf/` — downloadable PDF versions, when available
- `assets/` — supporting images, diagrams, or screenshots
